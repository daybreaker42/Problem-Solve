피로도 일일 최댓값
-
T <= 10^6이고
N, M <= 50
-> 다 50^2개씩 다르면 
total = 50^4^10^6 = 5*10*10
-> 근데 그럴수가 없음.
n*m칸에서 나와 k개 다른 진법의 개수는 nmCk -> n=50, m=50일때 2500Ck = 2500C2500-k인데
이때 k=2500이면 1개 다른건 총 1개만 존재
k = 2499면 2500개 존재
...
k = 1250일때 2500C1250으로 최대. (2500C2가 이미 3백만을 넘어서 T<=10^6 범위에선 2개만 다른것도 하루에 다 다른걸로 배치가능)
두 진법은 최대 1250개 다를 수 있으므로
-> 1250개 다를때 피로도 = 50^4/4 = 1562500 ~= 1.5*10^6

- 하루 최대 피로도
=> TiredMaxPerDay = Tmax * Tiredmax = 10^6*1.5*10^6 ~= 1.5*10*12 ~= 1.5조

- 최대 피로도 합
-> Lmax * TiredMaxPerDay = 10^6*1.5*10^12 = 1.5*10^18
