# x와 k
- 백준 1322
- 240519 solved

## 생각
1. x + y = x | y 에서 x를 이진수로 변환해서 x의 0인 부분에 y가 들어가면 or이랑 +이랑 같다는 점을 이용
    1. x와 k를 이진수로 변환
    2. 새로 bitset을 하나 만들고, x의 0의 자리마다(일의자리부터 올라가면서) k의 각 자리를 집어넣기
    3. 해당 bitset을 정수로 변환해 출력.
- 이렇게 되는 이유

ex) 5 5일때
5 => 00000101(2)
이때, x + y = x | y 이려면 x의 0부분에 y의 모든 이진수 부분이 들어가야 함.
이때 k = 101(2)이므로, x의 1, 3, 4번째 인덱스(뒤에서부터)에 이를 넣으면
x + y = x | y = 00010101(2)이 된다.
이때의 y는 10010(2) = 18이다.
