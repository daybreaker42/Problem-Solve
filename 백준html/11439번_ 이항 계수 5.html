11439번 - 이항 계수 5

 
시간 제한메모리 제한제출정답맞힌 사람정답 비율
1 초 256 MB108647229646.322%

문제

자연수 N\(N\)과 정수 K\(K\)가 주어졌을 때 이항 계수 (NK)\(\binom{N}{K}\)를 M\(M\)으로 나눈 나머지를 구하는 프로그램을 작성하시오.

입력

첫째 줄에 N\(N\), K\(K\)와 M\(M\)이 주어진다. (1 ≤ N\(N\) ≤ 4×106, 0 ≤ K\(K\) ≤ N\(N\), 2 ≤ M\(M\) ≤ 4×106)

출력

 (NK)\(\binom{N}{K}\)를 M\(M\)으로 나눈 나머지를 출력한다.

제한

예제 입력 1 

5 2 3

예제 출력 1 

1

예제 입력 2 

30 3 3

예제 출력 2 

1

예제 입력 3 

100 45 13

예제 출력 3 

2

예제 입력 4 

100 46 72

예제 출력 4 

48

예제 입력 5 

50 23 9

예제 출력 5 

1
